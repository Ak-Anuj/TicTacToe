# Tic-Tac-Toe

A simple, browser-based Tic-Tac-Toe game built using **HTML**, **CSS**, and **JavaScript**.

---

## 🔎 Description

This is a classic 3×3 Tic-Tac-Toe game for two players (local, same device). It includes win/draw detection, a visual board, and a reset option to start a new game.

---

## ⭐ Features

* Two-player gameplay (X vs O)
* Detects wins and draws
* Highlights winning combination
* Reset / New Game button
* Simple, responsive layout using CSS

---

## ▶️ How to Play

1. Open `index.html` in your browser.
2. Players take turns clicking an empty cell to place their mark (X or O).
3. The game automatically checks for a win or a draw after each move.
4. Click **Reset** (or **New Game**) to clear the board and start over.

---

## 🛠️ Installation / Run Locally

1. Clone or download the repository to your machine.
2. Open the project folder.
3. Double-click `index.html` or open it in your browser (e.g., `Ctrl+O` → select `index.html`).

No server or build tools required.

## 🧩 Implementation Notes

* `index.html` contains the markup for the 3×3 grid and controls.
* `styles.css` handles layout, responsive styles, and the winning highlight.
* `script.js` contains game logic: player turn management, win/draw checks, and UI updates.
